# pixi-napari-devbio

Pixi workspace to set up napari-devbio for pixel classification and cluster plotter analysis.

Created 19.12.2025 for LSM Image Analysis course, Universidad Mayor, Chile.

Install pixi https://pixi.sh/latest/ Create a Pixi folder and run a terminal there. Run `pixi init napari-devbio` `cd napari-devbio`

Replace the empty toml file in that directory with the toml file in this repository?

Run `pixi `install

Run `pixi run `assistant
